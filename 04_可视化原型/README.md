# 董小姐观点可视化原型

本目录存放本地可直接打开的 HTML 原型，用于查看和复盘董小姐观点拆解结果。

## 当前基准版

- `董小姐观点_美化版.html`：当前推荐使用版本。保留 Excel 骨架分类，默认只展示有数据分类，支持展开完整骨架、查看历史观点和原文摘录。

## 其他原型

- `董小姐观点_表格版.html`：表格版过渡原型，保留较多调试痕迹。
- `董小姐观点展示.html`：按分类展开的列表展示原型。
- `董小姐观点追踪.html`：按标签和日期追踪主题变化的矩阵原型。

## 使用方式

1. 直接用浏览器打开 `董小姐观点_美化版.html`。
2. 点击 `上传 Excel`，选择包含 A-K 字段的候选表。
3. 字段要求：经济体、维度、板块、议题/指标、子项、细项、原文摘录、精简观点、态度、标签、时间。

## 颜色规则

- 积极/利好：红色。
- 负面/利空：绿色。
- 谨慎：橙色。
- 中性：灰色。

## 同步边界

本目录只同步原型 HTML 和说明文档；原始截图、OCR 输出、每日候选表和 Excel 文件不进入 GitHub。
