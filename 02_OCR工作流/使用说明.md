# 截图正文自动整理

把课程长截图放进 `待整理截图` 文件夹，然后双击：

- `自动整理截图.command`：批量整理一次，输出到 `整理结果`
- `自动监控截图.command`：持续监控，每 30 秒自动整理新增或修改过的截图
- `拖拽图片到这里整理.command`：把图片文件或整个文件夹拖到脚本上，立即整理

默认同时输出：

- 每篇一个 `*_正文.txt`
- 每篇一个 `*_正文.md`
- 多篇时再生成 `全部正文_合并.txt` 和 `全部正文_合并.md`

默认只保留真正正文，从“每天10分钟……”开始，不保留优惠券、常见问题、评论区。

董小姐观点每日流程里，OCR 工具只作为临时加工区。正式输出请写到：

`/Users/a123/Downloads/董小姐/03_整理输出/YYYY-MM-DD`

当天原图、正文 `.txt`、正文 `.md`、Excel 候选表组成每日输出包；OCR 工作区里的截图用完后清空。

命令行用法：

```bash
cd /Users/a123/Downloads/董小姐/02_OCR工作流
python3 extract_article_text.py --folder 待整理截图
```

如果想保留“今日解读要点 / 优惠券 / 常见问题”，加：

```bash
python3 extract_article_text.py --folder 待整理截图 --include-meta
```

只想输出 Markdown：

```bash
python3 extract_article_text.py --folder 待整理截图 --format md
```

也可以直接给图片路径：

```bash
python3 extract_article_text.py /Users/a123/Downloads/IMG_4460.JPG /Users/a123/Downloads/IMG_4398.JPG
```

也可以直接给文件夹路径：

```bash
python3 extract_article_text.py /Users/a123/Downloads/课程截图
```
